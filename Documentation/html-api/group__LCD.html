Detailed Description

This module provides an interface to control a 4-bit HD44780-based LCD connected to the LPC1769 microcontroller.

It includes functions for writing characters, strings, controlling the cursor, and managing display settings.

The LCD data pins are connected to the LPC1769 microcontroller as described in the table below:

LCD LPC1769
D0 .. D3 Not connected
D4 .. D7 P2.0 .. P2.3
EN P0.11
RS P0.10
WR GND
